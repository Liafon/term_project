Topic:飲食追蹤
Basic feature:
1.設定基本信息（如年齡、體重、身高），系統根據這些信息生成建議的營養攝取量

2.可以記錄自己的飲食，系統根據食物的營養成分給出建議

Advanced Features:

1.增加考量運動對飲食的影響

使用方式:
依序輸入姓名、年齡、體重、身高、運動程度
